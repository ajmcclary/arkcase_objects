Accessibility

Keyboard navigation and screen reader support for Person cards.

Keyboard Navigation

  • Tab: Navigate between person cards
  • Enter/Space: Open person profile or actions
  • Arrow Keys: Navigate within contact options
  • Escape: Close dialogs or menus

Screen Reader Support

  • Semantic HTML with proper heading hierarchy
  • ARIA labels for contact type and status
  • Profile images have meaningful alt text
  • Contact actions clearly labeled for assistive technology
